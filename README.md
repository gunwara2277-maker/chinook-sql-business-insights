# Chinook SQL Business Insights

This project is a SQL-based business analysis using the **Chinook sample database**.  
The objective is to demonstrate practical SQL skills by answering real-world business questions related to sales, customers, and employees.

The project focuses on **data analysis and business insights**, rather than complex SQL syntax, making it suitable for roles such as Product Manager (Tech), Business Analyst.

---

## Dataset

- **Database:** Chinook (SQLite)
- **Source:** SQLite official tutorial
- **Domain:** Digital music store sales
- **Main Tables Used:** `invoices`, `customers`, `employees`

The Chinook database is widely used for learning and demonstrating SQL analysis skills in a business context.

---

## Business Questions & SQL Analysis

All SQL queries are stored in the `queries/` folder.

### Query 1: Top Countries by Invoice Count
- Identify which countries generate the highest number of invoices.

**File:** `01_invoices_by_country.sql`

---

### Query 2: Revenue by Country
- Calculate total revenue generated by each country.
- Useful for understanding geographic revenue distribution.

**File:** `02_revenue_by_country.sql`

---

### Query 3: Purchase Frequency per Customer
- Analyze how many times each customer made a purchase.
- Helps identify repeat customers.

**File:** `03_purchase_frequency_by_customer.sql`

---

### Query 4: Top Customers by Revenue
- Identify the customers who generate the highest total revenue.
- Useful for customer value and retention analysis.

**File:** `04_top_customers_by_revenue.sql`

---

### Query 5: Invoice Value Segmentation
- Segment invoices into value tiers:
  - **A:** Low value
  - **AA:** Medium value
  - **AAA:** High value
- Analyze invoice distribution and revenue contribution by segment.

**File:** `05_invoice_value_segmentation.sql`

---

### Query 6: Customers per Employee
- Analyze how many customers each employee is responsible for.
- Useful for workload distribution and support analysis.

**File:** `06_customers_per_employee.sql`
